# Data Files Guide

## CSV Source Files (Included in Git)

These CSV files are the source data for the workshop and are committed to the repository:

### `samples/inspection_logs.csv`
- **Records:** 20 inspection records
- **Generates:** 20 Inspection Certificate PDFs
- **Fields:** inspection_id, site, production_line, supplier, part_number, part_description, inspection_date, inspector, result, measurement_value, measurement_unit, spec_min, spec_max, notes
- **Generator:** `scripts/generate_inspection_pdfs.py`
- **Output:** `data/raw/pdf/inspections/INS-2024-XXX.pdf`

### `samples/maintenance_logs.csv`
- **Records:** 15 maintenance event records
- **Generates:** 15 Maintenance Work Order PDFs
- **Fields:** event_id, site, machine_id, machine_description, event_type, event_date, downtime_hours, technician, description, parts_replaced, notes
- **Generator:** `scripts/generate_maintenance_pdfs.py`
- **Output:** `data/raw/pdf/maintenance/MNT-2024-XXX.pdf`

### `samples/ncr_reports.csv`
- **Records:** 12 basic NCR records
- **Fields:** ncr_id, linked_inspection_id, site, supplier, part_number, part_description, severity, status, description, root_cause, corrective_action, opened_at, reviewed_at, closed_at
- **Purpose:** Simple NCR data for CSV ingestion testing

### `samples/ncr_detailed.csv`
- **Records:** 50 comprehensive NCR records
- **Generates:** 50 NCR Report PDFs (4 pages each)
- **Fields:** ncr_id, title, reference, status, revision, creator, department_creator, updated_by, updated_date, location, description, initial_analysis, data_significant, date_occurrence, cause_of_ncr, close_out_date, reason_closure, disposition, person_responsible, department_responsible, nrb_participant_email, nrb_participant_name, nrb_participant_type, decision_date, decision_type, classification_initiator, classification_confirmer, classification_prima, linked_inspection_id, site, supplier, part_number, severity, opened_at
- **Generator:** `scripts/generate_ncr_pdfs.py`
- **Output:** `data/raw/pdf/ncr/NCR-2024-XXX.pdf`

## Generated Files (Excluded from Git)

These files are generated by scripts and are NOT committed to git:

### PDF Documents
- `data/raw/pdf/ncr/*.pdf` - 50 NCR reports (548KB total for all PDFs)
- `data/raw/pdf/inspections/*.pdf` - 20 inspection certificates
- `data/raw/pdf/maintenance/*.pdf` - 15 maintenance work orders

### CSV Files in raw/
- `data/raw/*.csv` - Copies of sample CSV files for ingestion testing
- These are excluded because they're duplicates of the samples

## Quick Reference

| CSV File | Records | PDF Output | Generator Command |
|----------|---------|------------|-------------------|
| `ncr_detailed.csv` | 50 | NCR Reports (4 pages) | `make gen-ncr` |
| `inspection_logs.csv` | 20 | Inspection Certificates | `make gen-inspect` |
| `maintenance_logs.csv` | 15 | Maintenance Work Orders | `make gen-maint` |

## Generating PDFs

```bash
# Generate all PDFs (85 total)
make gen-pdfs

# Or individually
make gen-ncr       # 50 NCR reports
make gen-inspect   # 20 Inspection certificates
make gen-maint     # 15 Maintenance work orders
```

## Workshop Usage

1. **Session 3-4:** Copy sample CSVs to `data/raw/` for ingestion testing
2. **Session 5:** Query database to see ingested data
3. **Advanced:** Generate PDFs to test document ingestion and linking
